# Job Tracker Client

A React frontend for tracking job applications. Built with Vite, Tailwind CSS, and React Router.

## 🚀 Live Demo
https://your-vercel-url.vercel.app

## 🛠 Tech Stack
- React + Vite
- Tailwind CSS v4
- React Router v6
- Axios
- Context API

## ✨ Features
- Register and login with JWT auth
- Dashboard with application stats
- Add, edit, and delete job applications
- Search by company or position
- Filter by application status
- Protected routes
- Responsive design

## 🏃 Run Locally

1. Clone the repo
   git clone https://github.com/your-username/job-tracker-client

2. Install dependencies
   npm install

3. Create a .env file
   VITE_API_URL=http://localhost:5000/api

4. Start the dev server
   npm run dev
